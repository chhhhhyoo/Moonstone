# Handoff Contract

## Required Handoff Package

1. `patch.diff`
2. `checks.md`
3. `intent.md`
4. `traceability.md`

## Minimum Evidence

`checks.md` must include:

1. executed commands
2. pass/fail result
3. unresolved risks or explicit "none"

## Rejection Conditions

1. missing handoff artifact
2. weakened mandatory gates
3. public contract changes without spec/test/spec-impact updates
4. unverifiable success claims
